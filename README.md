# SimpleComicsViewer

## This Program is so buggy.

#### Depend
- GTK3
- GDK3
- zlib

#### LICENCE
GPLv3

#### Feature
- Load Image
- Load Compressed File
- Show Spread

#### Supported file format
- Single File
    - JPEG
    - PNG
- Compressed File
    - PNG or JPEG at non encrypted zip file.

#### Manual
- Shortcut Key
    - Move to right
        l or right arrow or Ctrl+f
    - Move to left
        h or left arrow or Ctrl+b
    - Open File
        Ctrl+o
    - Quit
        Ctrl+q or Alt+F4
    - Change Single to Sprad
        Ctrl+d
    - Change Spread to Single
        Ctrl+s
